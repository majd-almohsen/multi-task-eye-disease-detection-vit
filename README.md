# multi-task-eye-disease-detection
A multi-task deep learning system based on Vision Transformers for retinal disease classification and diabetic retinopathy severity assessment.
This project aims to develop a multi-task intelligent system for retinal disease diagnosis based on Vision Transformers (ViT). The proposed system analyzes fundus images and performs two main tasks simultaneously.

The first task focuses on disease classification into three primary categories: normal cases, diabetic retinopathy, and other ocular diseases. The second task is dedicated to assessing the severity of diabetic retinopathy using a binary classification approach, dividing cases into mild-to-moderate and severe stages.

The model leverages a pre-trained Vision Transformer to extract deep visual features from retinal images, followed by specialized classification heads for each diagnostic task. This approach enables efficient multi-task learning while maintaining high performance in medical image analysis.

Ultimately, this project aims to provide a clinical decision support tool that assists ophthalmologists by delivering comprehensive diagnostic insights and enabling early detection of eye diseases.
